# 劳务报酬个税计算器

一款基于 Tauri 构建的桌面应用，用于计算劳务报酬所得个人所得税，支持正向测算、反向推算、对比分析及年度汇算功能。

## 功能特性

### 1. 正向测算
根据每月税前收入，使用累计预扣法计算应缴个人所得税。
- 支持 1-12 月份的个性化收入输入
- 提供统一收入模式，快速设置所有月份相同收入
- 图表展示每月税额及累计税额变化趋势
- 导出计算结果为 CSV 文件

### 2. 反向推算
根据目标税后收入，反向推算所需税前收入。
- 支持多月份批量设置
- 提供验证功能，确保推算结果准确
- 显示对应的税率档次信息

### 3. 对比分析
对比累计预扣法与一般预扣法的税额差异，分析节税空间。
- 并排展示两种方法的计算结果
- 图表化呈现税额对比
- 计算节税金额及节税比例

### 4. 年度汇算
年度个人所得税汇算清缴计算。
- 支持专项扣除（三险一金等）
- 支持专项附加扣除（子女教育、赡养老人、住房租金、继续教育等）
- 支持其他法定扣除
- 自动计算应补税或应退税金额

## 技术架构

### 前端技术栈
- **React 18** - 用户界面框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **TailwindCSS** - 样式框架
- **Recharts** - 图表可视化
- **Lucide React** - 图标库

### 后端技术栈
- **Tauri 2** - 桌面应用框架
- **Rust** - 高性能计算引擎

### 核心计算模块（tax_engine.rs）
- 累计预扣法计算 (`compute_cumulative`)
- 一般预扣法计算 (`compute_general`)
- 反向推算算法 (`reverse_compute`)
- 年度汇算计算 (`compute_annual_settlement`)

## 税率规则

### 累计预扣法税率表（月度累计）
| 应纳税所得额 | 税率 | 速算扣除数 |
|-------------|------|-----------|
| ≤ 36,000 | 3% | 0 |
| ≤ 144,000 | 10% | 2,520 |
| ≤ 300,000 | 20% | 16,920 |
| ≤ 420,000 | 25% | 31,920 |
| ≤ 660,000 | 30% | 52,920 |
| ≤ 960,000 | 35% | 85,920 |
| > 960,000 | 45% | 181,920 |

### 一般预扣法税率表（单次收入）
| 收入额 | 税率 | 速算扣除数 |
|-------|------|-----------|
| ≤ 20,000 | 20% | 0 |
| ≤ 50,000 | 30% | 2,000 |
| > 50,000 | 40% | 7,000 |

### 扣除规则
- **费用减除**：收入额的 20%
- **每月减除费用**：5,000 元
- **年度减除费用**：60,000 元

## 开发环境

### 前置要求
- Node.js >= 18
- Rust >= 1.70

npm install

```bash

pnpm tauri build
```

## 使用说明

### 正向测算示例
1. 选择"正向测算"功能
2. 输入各月税前收入（或使用统一收入模式）
3. 点击"计算"按钮
4. 查看计算结果及图表
5. 可导出 CSV 报表

### 反向推算示例
1. 选择"反向推算"功能
2. 设置目标税后收入和月份数量
3. 点击"反向推算"按钮
4. 系统自动计算所需税前收入
5. 可查看验证结果确保准确性

### 年度汇算示例
1. 选择"年度汇算"功能
2. 输入全年劳务报酬总收入
3. 填写专项扣除、专项附加扣除等
4. 点击"计算年度汇算"
5. 查看应补税或应退税金额

## 数据说明

### 累计预扣法
适用于居民个人从多处取得劳务报酬的情形，按月份累计计算，年度并入综合所得计税。

### 一般预扣法
适用于单次或少量劳务报酬收入，按次或按月单独计算。

### 专项附加扣除
- 子女教育：每个子女 1000 元/月
- 赡养老人：独生子女 2000 元/月，非独生子女分摊
- 住房租金：城市不同，800-1500 元/月
- 继续教育：学历教育 400 元/月，职业继续教育 3600 元/年

1. 本计算器仅供参考，实际税额以税务机关核定为准
2. 专项附加扣除需符合国家最新政策规定
3. 年度汇算需在规定时间内完成申报
4. 计算结果精确到小数点后两位
